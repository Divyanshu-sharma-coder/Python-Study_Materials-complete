# Hybrid and Hierarchical Inheritance in Python

This tutorial covers the remaining two major categories of Object-Oriented Programming (OOP) inheritance in Python:

- **Hybrid Inheritance**
- **Hierarchical Inheritance**

---

## 1. Hybrid Inheritance

Hybrid inheritance is a combination of two or more types of inheritance models (single, multiple, multilevel, or hierarchical) used together in a single program structure.

---

## Conceptual Implementation

```python
class BaseClass:
    pass


class Derived1(BaseClass):
    # Single inheritance example
    pass


class Derived2(BaseClass):
    pass


class Derived3(Derived1, Derived2):
    # Multiple inheritance example
    pass
```

---

## Architectural Breakdown

### ✔ Single / Multilevel Part

```
BaseClass → Derived1
BaseClass → Derived2
```

### ✔ Multiple Inheritance Part

```
Derived3 inherits from:
Derived1 + Derived2
```

---

## ✔ Why it is called Hybrid Inheritance?

Because it combines:

- Single Inheritance
- Multiple Inheritance
- (and often Multilevel patterns too)

So the structure becomes a **mixed inheritance model**.

---

## 2. Hierarchical Inheritance

Hierarchical inheritance occurs when **multiple child classes inherit from a single parent class**.

---

## Corporate Analogy

```
                CEO (BaseClass)
                     │
     ┌───────────────┼───────────────┐
     ▼               ▼               ▼
 Manager1        Manager2        Manager3
```

All managers share common features from the CEO but work independently.

---

## Conceptual Implementation

```python
class BaseClass:
    pass


# Multiple child classes from one parent
class D1(BaseClass):
    pass

class D2(BaseClass):
    pass


# Further branching
class D3(D1):
    pass

class D4(D1):
    pass
```

---

## 3. Visual Overview of All Inheritance Types

### ✔ Single Inheritance

```
A → B
```

---

### ✔ Multiple Inheritance

```
A + B → C
```

---

### ✔ Multilevel Inheritance

```
A → B → C
```

---

### ✔ Hierarchical Inheritance

```
        A
       / \
      B   C
     / \
    D   E
```

---

### ✔ Hybrid Inheritance

```
A → B → C
     ↑   ↑
     └───┘
```

(Mixed structure of multiple patterns)

---

## 4. Key Takeaways

✔ Hybrid inheritance = combination of multiple inheritance types  
✔ Hierarchical inheritance = one parent, many children  
✔ Python supports all inheritance types simultaneously  
✔ MRO is important in hybrid/multiple structures  

---

## Final Idea

- **Hierarchical = one → many**
- **Hybrid = mixed inheritance system**
- Together, they help design complex real-world systems in OOP
