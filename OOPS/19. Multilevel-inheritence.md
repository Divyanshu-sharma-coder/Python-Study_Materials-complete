# Multilevel Inheritance in Python

## 1. What is Multilevel Inheritance?

Multilevel inheritance is a type of inheritance in Object-Oriented Programming (OOP) where a **derived class inherits from another derived class**. This creates a **chain-like relationship**:

```
Grandparent → Parent → Child
```

Each level automatically inherits properties from all previous levels.

---

## Analogy

- **Class A (Animal)** → Grandparent  
- **Class B (Dog)** → Parent  
- **Class C (GoldenRetriever)** → Child  

So the last class gets features from *both previous classes*.

---

## Structure Diagram

```
┌────────────────────┐
│   Base Class       │  (Animal)
└─────────┬──────────┘
          │
          ▼
┌────────────────────┐
│ Derived Class 1    │  (Dog)
└─────────┬──────────┘
          │
          ▼
┌────────────────────┐
│ Derived Class 2    │  (GoldenRetriever)
└────────────────────┘
```

---

## 2. Key Benefits

✔ **Code Reusability**  
- No need to rewrite logic in every class

✔ **Clean Architecture**  
- Code becomes structured and hierarchical

✔ **DRY Principle**  
- Avoids repetition (Don't Repeat Yourself)

---

## 3. Practical Code Implementation

```python
class Animal:
    def __init__(self, name, species):
        self.name = name
        self.species = species

    def show_details(self):
        print(f"Name: {self.name}")
        print(f"Species: {self.species}")


class Dog(Animal):
    def __init__(self, name, breed):
        Animal.__init__(self, name, species="Dog")
        self.breed = breed

    def show_details(self):
        Animal.show_details(self)
        print(f"Breed: {self.breed}")


class GoldenRetriever(Dog):
    def __init__(self, name, color):
        Dog.__init__(self, name, breed="Golden Retriever")
        self.color = color

    def show_details(self):
        Dog.show_details(self)
        print(f"Color: {self.color}")


# Creating object
o = GoldenRetriever("Tommy", "Black")
o.show_details()
```

---

## 4. Code Execution Flow

When you call:

```python
o.show_details()
```

Execution flow happens like this:

### Step-by-step flow

```
GoldenRetriever.show_details()
        ↓
Dog.show_details()
        ↓
Animal.show_details()
        ↓
Back to Dog (prints breed)
        ↓
Back to GoldenRetriever (prints color)
```

---

## 5. Output

```text
Name: Tommy
Species: Dog
Breed: Golden Retriever
Color: Black
```

## 6. Method Resolution Order (MRO)

You can check how Python searches methods:

```python
print(GoldenRetriever.mro())
```

---

## MRO Output

```
[<class '__main__.GoldenRetriever'>,
 <class '__main__.Dog'>,
 <class '__main__.Animal'>,
 <class 'object'>]
```

---

## 7. Key Takeaways

✔ Multilevel inheritance = chain of classes  
✔ Each class builds on the previous one  
✔ Methods are resolved using **MRO**  
✔ Helps build structured OOP systems  

---

## Final Idea

Multilevel inheritance creates a **layered hierarchy**, where each child class extends functionality step by step instead of redefining everything again.
